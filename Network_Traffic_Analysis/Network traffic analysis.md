# Network Traffic Analysis

This folder contains write-ups and methodology guides for NCL Network Traffic Analysis challenges.

## What is Network Traffic Analysis?

Network Traffic Analysis challenges require you to inspect packet captures (PCAPs) or live network data to reconstruct sessions, identify malicious traffic, extract files, or uncover hidden information.

## Common Tools

- Wireshark
- tcpdump
- tshark
- NetworkMiner
- Zeek (formerly Bro)
- Scapy (Python)

## Contributing

Create a folder for each challenge (e.g., `Challenge Name (Difficulty)/`) and add your write-up following the [`CHALLENGE_TEMPLATE.md`](../templates/CHALLENGE_TEMPLATE.md).

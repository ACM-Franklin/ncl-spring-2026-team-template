# Cryptography

This folder contains write-ups and methodology guides for NCL Cryptography challenges.

## What is Cryptography?

Cryptography challenges test your ability to decrypt, encode/decode, or crack ciphers. You may encounter classical ciphers, modern encryption schemes, or custom encoding algorithms.

## Common Tools

- CyberChef
- hashcat
- openssl
- Python (pycryptodome)
- dcode.fr
- quipqiup (for substitution ciphers)

## Contributing

Create a folder for each challenge (e.g., `Challenge Name (Difficulty)/`) and add your write-up following the [`CHALLENGE_TEMPLATE.md`](../templates/CHALLENGE_TEMPLATE.md).

# Scanning & Reconnaissance

This folder contains write-ups and methodology guides for NCL Scanning & Reconnaissance challenges.

## What is Scanning & Recon?

Scanning & Recon challenges require you to enumerate hosts, open ports, running services, and potential vulnerabilities on a target system or network using active and passive scanning techniques.

## Common Tools

- Nmap
- Masscan
- Nikto
- gobuster / dirb / feroxbuster
- Shodan
- Netcat

## Contributing

Create a folder for each challenge (e.g., `Challenge Name (Difficulty)/`) and add your write-up following the [`CHALLENGE_TEMPLATE.md`](../templates/CHALLENGE_TEMPLATE.md).

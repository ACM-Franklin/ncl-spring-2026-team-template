# Forensics

This folder contains write-ups and methodology guides for NCL Forensics challenges.

## What is Forensics?

Forensics challenges test your ability to recover, analyze, and interpret digital artifacts. This includes disk images, memory dumps, steganography, file carving, and metadata analysis.

## Common Tools

- Autopsy / Sleuth Kit
- Volatility (memory forensics)
- binwalk / foremost (file carving)
- ExifTool (metadata)
- steghide / zsteg / stegsolve (steganography)
- strings / xxd / hexdump

## Contributing

Create a folder for each challenge (e.g., `Challenge Name (Difficulty)/`) and add your write-up following the [`CHALLENGE_TEMPLATE.md`](../templates/CHALLENGE_TEMPLATE.md).

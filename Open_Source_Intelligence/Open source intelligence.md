# Open Source Intelligence (OSINT)

This folder contains write-ups and methodology guides for NCL Open Source Intelligence challenges.

## What is OSINT?

OSINT challenges test your ability to gather information from publicly available sources. Common tasks include locating individuals, organizations, or artifacts using search engines, social media, public records, and other open sources.

## Common Tools

- Google Dorking
- Shodan
- Maltego
- theHarvester
- Recon-ng
- Wayback Machine / web.archive.org

## Contributing

Create a folder for each challenge (e.g., `Challenge Name (Difficulty)/`) and add your write-up following the [`CHALLENGE_TEMPLATE.md`](../templates/CHALLENGE_TEMPLATE.md).

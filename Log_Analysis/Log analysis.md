# Log Analysis

This folder contains write-ups and methodology guides for NCL Log Analysis challenges.

## What is Log Analysis?

Log Analysis challenges require you to parse and investigate log files—such as web server logs, firewall logs, or system logs—to identify malicious activity, anomalies, or specific events.

## Common Tools

- grep / awk / sed
- Splunk
- ELK Stack (Elasticsearch, Logstash, Kibana)
- Python (pandas, re)
- GoAccess

## Contributing

Create a folder for each challenge (e.g., `Challenge Name (Difficulty)/`) and add your write-up following the [`CHALLENGE_TEMPLATE.md`](../templates/CHALLENGE_TEMPLATE.md).

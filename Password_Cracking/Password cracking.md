# Password Cracking

This folder contains write-ups and methodology guides for NCL Password Cracking challenges.

## What is Password Cracking?

Password Cracking challenges present hashed or encoded credentials that you must recover. You may use wordlists, rule-based attacks, brute force, or rainbow tables depending on the hash type and complexity.

## Common Tools

- hashcat
- John the Ripper
- CyberChef
- rockyou.txt and custom wordlists
- hash-identifier

## Contributing

Create a folder for each challenge (e.g., `Challenge Name (Difficulty)/`) and add your write-up following the [`CHALLENGE_TEMPLATE.md`](../templates/CHALLENGE_TEMPLATE.md).
